Detailed Description

template<size_t BUFFER_SIZE, size_t MAX_TOKENS>
class JsonParserStatic< BUFFER_SIZE, MAX_TOKENS >

Creates a JsonParser with a static buffer.

You normally use this when you're creating a parser as a global variable. For small data (under around 256 bytes so) you can also allocate one on the stack.

Parameters
BUFFER_SIZEThe maximum size of the data to be parsed, in bytes. If you are parsing a webhook response split into parts, this is the total size of all parts.
MAX_TOKENSThe maximum number of tokens you expect. Each object has a token and two for each key/value pair. Each array is a token and one for each element in the array.
